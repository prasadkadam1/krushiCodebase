Translation Steps

  • Request and understand the script before translation.
  • Use simple, clear language suited for farmers.
  • Type the translated script clearly with double-spacing.
  • Maintain timing and style, using punctuation for clarity.
  • Have the script checked by an expert and test it aloud.

Voice Recording Best Practices

  • Record in a quiet place with good acoustics.
  • Use WAV 48KHz 16bit for high-quality audio.
  • Keep a clear, conversational tone and check levels to avoid distortion.
  • Label files properly with programme, language, and shot numbers.
  • Review all recordings before finalizing.
